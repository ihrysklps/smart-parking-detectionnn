# 🚗 Smart Parking Detection System

## 📌 Overview

This project detects available and occupied parking spaces using computer vision techniques with OpenCV.

## 🎯 Problem

Finding parking in crowded areas is time-consuming. This system helps identify free parking spots automatically.

## 🚀 Features

* Detects parking occupancy
* Works with images and video
* Displays available slots in real-time

## 🛠️ Tech Stack

* Python
* OpenCV
* NumPy

## ⚙️ Setup

```bash
git clone https://github.com/your-username/smart-parking-detection
cd smart-parking-detection
pip install -r requirements.txt
```

## ▶️ Run

```bash
python main.py
```

## 📷 How It Works

* Define parking slots manually
* System checks pixel density
* Displays:

  * Green → Empty
  * Red → Occupied

## 📊 Output

Shows number of free parking slots on screen.
